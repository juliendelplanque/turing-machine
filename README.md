# TuringMachine
A Turing Machine simulator written in Pharo.

To load this package juste execute the following code in a workspace:
~~~
Metacello new
    baseline: 'TuringMachine';
    repository: 'github://juliendelplanque/turing-machine/repository';
    load.
~~~
